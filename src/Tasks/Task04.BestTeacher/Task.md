# Задача 04: Лучший преподаватель

Двумерный массив `double[N, M]` — оценки N преподавателей от M студентов (оценки 0–5).

Найти лучшего преподавателя: строку с максимальным **усечённым средним**
(из расчёта исключается одна наибольшая и одна наименьшая оценка).

Функция возвращает индекс строки и вычисленное среднее.

**Пример:** для тестового массива ответ — индекс 3, среднее 4.15.

**Сложность:** O(N×M) по времени, O(1) по памяти.
