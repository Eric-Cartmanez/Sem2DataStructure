# Задача 06: Поиск в 2D-массиве

На вход подаётся двумерный массив `int[,]`, элементы которого отсортированы по неубыванию
построчно (конец строки i < начало строки i+1).

Найти первое вхождение заданного значения. Алгоритм должен быть эффективнее линейного поиска.

**Алгоритм:** бинарный поиск — 2D-массив трактуется как плоский 1D.
Индекс `mid` преобразуется в `[mid / cols, mid % cols]`.

**Сложность:** O(log(N×M)) по времени, O(1) по памяти.

**Примеры:**
- Массив `{{2,6,7,9,9,14},{18,20,26,26,29,40},{44,47,50,51,55,62}}`, поиск 18 → `[1, 0]`
- Тот же массив с 40 на позиции [1,4], поиск 40 → `[1, 4]`
