# Задача 07: Калейдоскоп

Вывести в консоль квадратный калейдоскоп из цветных блоков с симметрией
по горизонтали, вертикали и диагоналям. Размер (половина стороны) вводит пользователь (3–20).

**Правила генерации цветов:**
- Используется массив `colors = {0, 8, 7, 15, 14, 6, 12, 4, 5, 13, 11, 3, 9, 1, 2, 10}` (индексы ConsoleColor).
- Каждый следующий вычисляемый цвет отстоит от предыдущего на 1 или 2 позиции в этом массиве.
- Цвета генерируются только для 1/8 матрицы (верхний треугольник левого верхнего квадранта),
  остальное заполняется симметрично.
