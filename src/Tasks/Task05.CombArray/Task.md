# Задача 05: Комбинированный массив

На вход подаётся одномерный массив `int[]`. Необходимо вернуть новый массив,
содержащий только элементы, образующие неубывающую последовательность
(один проход, каждый элемент сравнивается с последним сохранённым).

**Пример:**
- Вход:  `{-6, -2, -3, 0, 6, 1, 3, 11, 11, 14}`
- Выход: `{-6, -2, 0, 6, 11, 11, 14}`

**Сложность:** O(n) по времени, O(n) по памяти.
