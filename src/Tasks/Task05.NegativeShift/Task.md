# Задача 05: Перестановка отрицательных

Переставить элементы одномерного массива `int[]` так, чтобы сначала шли
все **отрицательные** значения, затем все **неотрицательные** (≥0).

Порядок внутри групп не важен. Массив изменяется in-place.

**Алгоритм:** два указателя, левый накапливает позицию для следующего отрицательного.

**Сложность:** O(n) по времени, O(1) по памяти.
