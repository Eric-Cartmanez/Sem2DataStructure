# Задача 02: N-буквенные слова

Разработать программу, которая вводит натуральные числа M (≤33) и N,
и выводит все N-буквенные слова, состоящие из первых M букв русского алфавита.

Например, при N=3 и M=4 программа выводит 64 слова: ААА, ААБ, ..., ГГГ.

Задача решена двумя способами:
1. **Рекурсивный** — строим слово посимвольно, перебирая все буквы на каждом шаге.
2. **Итеративный** — каждое число i от 0 до M^N−1 трактуем как слово в M-ичной записи.
